# holbertonschool-web_react
# TypeScript Project

A series of tasks to learn TypeScript fundamentals.

## Tasks

- **task_0** - Basic types and interfaces: Student interface rendered in an HTML table
- **task_1** - Interfaces, Classes and functions: Teacher, Directors, and StudentClass
- **task_2** - Advanced types: Director/Teacher classes with type predicates and string literals
- **task_3** - Ambient namespaces: CRUD operations with type declarations
- **task_4** - Namespaces and declaration merging: Subjects with Teacher interface merging
- **task_5** - Brand convention and nominal typing: MajorCredits and MinorCredits interfaces

## Requirements

- TypeScript compiler should return no warnings or errors
- All variables use TypeScript when possible
- Tested with Jest 29
